This repository contains code and and example to convert government level observations into yearly data.
Something similar will work with any discreet time periods which are inconsisent.
File assigns government to a year if it is the one in power the longest that year.
The .dta files with 13 and Old on the end are for users that do not have STATA 14.
Datasets are abbreviated and intended only to demonstrate the code works. 
Additional annotation in .do file.
